# NSE Dividend Scraper

A lightweight Go script to scrape and extract upcoming equity dividend actions from the National Stock Exchange of India (NSE).

## Requirements

- Go 1.18+

## Usage

Navigate to the project directory and run the script:

```bash
cd NSE_scraper
go run main.go
```

The scraper fetches the latest corporate actions, filters for equity dividends, and outputs the extracted dividend amount.
